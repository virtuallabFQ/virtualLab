# Laboratório Virtual

<div align="justify">
O ensino de Física-Química em Portugal, particularmente ao nível do ensino secundário, depende fortemente da componente experimental para a consolidação dos conceitos ensinados em sala de aula. Contudo, a realidade logística de muitas instituições de ensino impõe barreiras ao acesso a laboratórios físicos devidamente equipados com instrumentos de precisão. 
Neste sentido, o projeto Laboratório Virtual surge com o objetivo de verificar a seguinte hipótese: será possível trazer às escolas sem acesso a um laboratório físico a capacidade de enriquecer o seu ensino com a vertente experimental?
</div>
